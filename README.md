#  Spring Boot Learning Repository

A structured, hands-on collection of Spring Boot demonstrations
organized topic by topic — from core concepts to advanced real-world
implementations.

---

## 👨‍💻 About This Repository

This repository is built with the goal of demonstrating each Spring Boot
concept in a clean, isolated, and runnable project. Every sub-module
contains only the code relevant to that specific concept — no clutter,
no confusion.

Each demonstration includes:
- Working source code
- Required dependencies (pom.xml)
- Configuration (application.properties)
- Explanation (README.md)

---
---

## 🚀 How to Run Any Demo

1. Clone the repository
```bash
   git clone https://github.com/GirirajGupta45/Spring-Boot-Learning-Repository.git
```

2. Navigate to any demo folder
```bash
   cd 01-Spring-Overview/01-actuator-demo
```

3. Run using Maven
```bash
   mvn spring-boot:run
```
